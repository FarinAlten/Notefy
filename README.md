#  Notefy

**Notefy** is a minimalistic and user-friendly notes app designed to make capturing, managing, and organizing your thoughts effortless.  
Perfect for quick ideas, to-do lists, or longer notes — all within an intuitive interface.

---

##  Features

-  **Quick Note Creation** – Capture new thoughts or tasks in seconds   
- **Dark Mode** – Comfortable on the eyes, day and night
